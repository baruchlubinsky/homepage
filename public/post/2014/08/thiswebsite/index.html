I built this website using Hugo. I found it a little tricky to get started, but it is very new software. Now that it is up and running, I’m really getting into it.

The styling use the Hyde theme for Hugo which is based on MDO’s package of the same name.

View the source here.

The website is hosted on S3. Uploading is simple with S3cmd: 

s3cmd sync -P –exclude="DS_Store" public/ s3://baruch.lubinsky.co.za
